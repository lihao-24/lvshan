# 闾山文化蜡烛品牌官网 - 项目概览

## 项目简介
闾山文化蜡烛品牌官方网站，展示品牌文化、产品价值和系列产品，传递东方美学与道家文化的品牌调性。

## 技术栈
- **前端**: 纯HTML/CSS/JavaScript
- **特点**: 无框架依赖，轻量级，易维护
- **交互**: 全屏滚动 + 轮播效果

## 核心特性

### 1. 全屏滚动体验
- 类似Apple产品页的沉浸式体验
- 每个section占满一屏
- 滚动自动吸附，类似PPT翻页
- 支持鼠标滚轮、触摸滑动、键盘方向键

### 2. 品牌故事轮播
- 5屏轮播展示品牌不同维度
- 自动播放 + 手动切换
- 精美背景图配合文案
- 淡入淡出过渡效果

### 3. 中英双语
- 完整的中英文内容
- 一键切换语言
- 语言选择持久化

### 4. 响应式设计
- 支持桌面端、平板端、移动端
- 优雅的移动端体验
- 图片自适应加载
