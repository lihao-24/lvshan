# 闾山文化蜡烛品牌官网 - 需求文档

## 1. 项目概述

### 1.1 项目目标
建设闾山文化蜡烛品牌的官方网站，展示品牌文化、产品价值和系列产品，传递东方美学与道家文化的品牌调性。

### 1.2 技术栈
- 纯HTML/CSS/JavaScript静态网站
- 响应式设计，支持移动端和桌面端
- 中英双语支持

## 2. 用户故事

### 2.1 品牌展示
**作为** 网站访问者  
**我想要** 通过轮播形式了解闾山文化的多个维度  
**以便** 全面理解品牌的文化底蕴和价值定位

**验收标准**:
- 1.1 首页采用全屏轮播形式，共5个轮播屏幕
- 1.2 轮播屏幕1展示品牌主视觉
  - 标题："Lvshan Tradition 闾山文化"
  - 副标题："以物载道 · 赠礼臻选"
  - 背景：武夷山脉云雾意境图
- 1.3 轮播屏幕2展示历史传承
  - 发源地：中国福建武夷山脉
  - 历史起源：唐代(公元713-741年)
  - 传承时间：1300余年
- 1.4 轮播屏幕3展示文化特色
  - 融合正统道教修行与闽越巫傩文化
  - 符箓仪轨被列入省级非物质文化遗产
- 1.5 轮播屏幕4展示核心理念
  - 道家生活美学
  - 将千年修行智慧凝练于日用之物
- 1.6 轮播屏幕5展示适合人群
  - 重视文化传承的友人
  - 追求心灵宁静的修行者
  - 喜爱东方美学的收藏家
  - 新居/节庆的祝福礼物
- 1.7 轮播自动播放（5秒/屏）
- 1.8 支持手动切换（左右箭头、底部指示点）
- 1.9 支持触摸滑动切换
- 1.10 轮播过渡效果流畅（淡入淡出）
- 1.11 所有核心内容提供中英双语版本

### 2.2 产品价值展示

**我想要** 了解产品的价值主张和原料信息  
**以便** 评估产品的品质和艺术价值

**验收标准**:
- 2.1 展示产品价值主张：传统文化工艺品定位、匠心手造、艺术价值与精神寓意
- 2.2 展示天然植物蜡(大豆蜡)原料信息
- 2.3 展示玻璃容器等其他原料

- 2.5 页面风格符合东方美学和高端礼品定位

## 3. 设计要求

### 3.1 整体风格
**验收标准**:
- 6.1 视觉风格统一、协调、不违和
- 6.2 体现东方美学 + 道家文化 + 现代简约的设计定位
- 6.3 传达文化传承、禅意、雅致、高端礼品的品牌调性
- 6.4 色彩方案符合道家美学（建议使用自然色调、墨色、金色等）
- 6.5 字体选择优雅，中文使用宋体或书法字体，英文使用衬线字体

### 3.2 响应式设计
**验收标准**:
- 7.1 网站在桌面端（1920px、1440px、1024px）正常显示
- 7.2 网站在平板端（768px）正常显示
- 7.3 网站在移动端（375px、414px）正常显示
- 7.4 图片在不同设备上自适应加载
- 7.5 文字在不同设备上可读性良好

### 3.3 交互体验
**验收标准**:
- 8.1 采用全屏滚动(Full-page Scrolling)效果，类似Apple产品页
- 8.2 每个section占满整个视口(100vh)
- 8.3 滚动时自动吸附到下一个section，类似PPT翻页
- 8.4 支持鼠标滚轮滚动切换
- 8.5 支持触摸滑动切换（移动端）
- 8.6 支持键盘方向键切换
- 8.7 右侧显示导航点，指示当前位置
- 8.8 导航点可点击跳转到对应section
- 8.9 滚动切换流畅，有适当的过渡动画
- 8.10 图片加载优化，使用懒加载技术
- 8.11 品牌故事轮播的悬停效果优雅
- 8.12 产品卡片的hover效果优雅
- 8.13 页面加载速度快（首屏加载时间 < 3秒）
- 8.14 滚动节流，防止快速滚动导致的问题

### 3.4 语言切换
**验收标准**:
- 9.1 提供中英文切换功能
- 9.2 语言切换按钮位置明显（建议在页面右上角）
- 9.3 切换语言后，所有内容同步更新
- 9.4 默认语言为中文
- 9.5 语言选择状态持久化（使用localStorage）

## 5. 技术要求

### 5.1 代码质量
**验收标准**:
- 12.1 HTML语义化，使用合适的标签
- 12.2 CSS使用BEM命名规范或类似的规范
- 12.3 JavaScript代码模块化，易于维护
- 12.4 代码注释清晰，关键功能有说明
- 12.5 无控制台错误或警告

### 5.2 性能优化
**验收标准**:
- 13.1 图片压缩优化，使用WebP格式（提供fallback）
- 13.2 CSS和JavaScript文件压缩
- 13.3 使用图片懒加载
- 13.4 关键CSS内联，非关键CSS异步加载
- 13.5 字体文件优化加载

### 5.3 浏览器兼容性
**验收标准**:
- 14.1 支持Chrome最新版本
- 14.2 支持Firefox最新版本
- 14.3 支持Safari最新版本
- 14.4 支持Edge最新版本
- 14.5 移动端支持iOS Safari和Android Chrome

### 5.4 SEO优化
**验收标准**:
- 15.1 每个页面有合适的title和meta description
- 15.2 使用语义化HTML标签（header, nav, main, section, article, footer）
- 15.3 图片有alt属性
- 15.4 使用Open Graph标签，支持社交媒体分享
- 15.5 提供robots.txt和sitemap.xml
